아래 지시를 Diagnostic Tools 페이지의 슬로건 섹션에만 적용하고,
다른 페이지나 기존 스크롤 구조(가로 → 세로 전환), 헤더 로직, 이미지 동작은 절대 수정하지 말 것.

1️⃣ 현재 문제 정의 (이해 확인용)

현재 슬로건 페이지에서 다음 문제가 발생하고 있다:

메인 텍스트, 설명 텍스트, “우리의 원칙” 박스가 동시에 화면에 존재하여 겹치거나 산만함

텍스트 등장 타이밍이 빠르고 겹쳐서 페이드업 효과가 저급하게 보임

텍스트 기준선이 명확하지 않아 고급스럽고 VIP 느낌이 사라짐

스크롤 시 텍스트들이 서로 다른 위치에서 튀듯 등장함

이 문제를 해결하는 것이 목적이다.

2️⃣ 수정 목표 (반드시 충족해야 함)

한 순간에 하나의 메시지 흐름만 인지되도록 구성

텍스트는 **위계(메인 → 설명 → 원칙)**가 명확해야 함

텍스트 겹침 ❌

과한 애니메이션 ❌

전체 톤은 Luxury / VIP / Calm / Minimal

3️⃣ 슬로건 섹션 구조 수정 지시 (HTML 구조)

슬로건 섹션 내부를 아래 3개의 그룹으로 명확히 분리한다:

🔹 그룹 1: 메인 텍스트

텍스트:

정확한 진단이 완치의 시작입니다


단일 문장

별도 wrapper 사용 (예: .slogan-main)

🔹 그룹 2: 설명 텍스트

텍스트:

서울본재활의학과는 최첨단 진단 장비에 아낌없이 투자합니다.
통증의 근본 원인을 빠르고 정확하게 찾아내어,
환자분에게 가장 적합한 치료 계획을 수립하기 위함입니다.


문단 wrapper 사용 (예: .slogan-description)

🔹 그룹 3: 우리의 원칙 카드

카드 형태의 독립 컴포넌트 (예: .principle-card)

내부 텍스트는 절대 하나의 덩어리로 묶지 말 것

반드시 아래 4개 요소로 분리:

타이틀

우리의 원칙


핵심 문장

모든 시술의 영상 가이드화


영문 서브 타이틀

Image-Guided Injection


설명 문단

감이나 경험에만 의존하는 '블라인드 주사'를 지양합니다.
모든 주사 치료 및 시술은 반드시 초음파나 C-arm 실시간
영상 장비를 보면서 진행합니다.

4️⃣ 초기 상태 설정 (중요)

페이지 진입 시 초기 상태는 반드시 아래와 같아야 한다:

메인 텍스트만 화면에 보임

설명 텍스트는 opacity 0

“우리의 원칙” 카드는 화면에 존재하지 않거나 완전히 숨김 상태

어떤 텍스트도 미리 공간을 차지하며 열려 있으면 안 됨

5️⃣ 등장 타이밍 및 애니메이션 규칙 (핵심)
🔹 ① 메인 텍스트

기준 위치: 화면 높이 기준 약 45vh

등장 효과:

opacity: 0 → 1

transform: translateY(16px) → 0

duration: 1.2초

easing: cubic-bezier(0.22, 0.61, 0.36, 1)

scale, blur, bounce 효과 절대 사용 금지

🔹 ② 설명 텍스트

메인 텍스트 등장 완료 후 0.4초 뒤

효과:

opacity: 0 → 1

이동 없음

duration: 1.0초

메인 텍스트 위치는 변하지 않음

🔹 ③ 우리의 원칙 카드 (스크롤 기반 등장)

메인 + 설명 텍스트가 완전히 고정된 상태에서

사용자가 추가로 스크롤을 한 번 더 내렸을 때 등장

카드 자체 효과

opacity: 0 → 1

scale: 0.98 → 1

duration: 1.3초

easing 동일

카드 내부 텍스트 등장 순서 (중요)

각 요소는 절대 동시에 등장하지 말 것

우리의 원칙

모든 시술의 영상 가이드화

Image-Guided Injection

설명 문단

각 요소 간 delay: 0.15~0.2초

효과: opacity만 변화

이동 ❌

scale ❌

6️⃣ 배치 및 정렬 규칙

모든 텍스트는 가로 중앙 정렬

단, 문단 텍스트는 left-align

최대 너비 제한:

메인 텍스트: 약 900px

원칙 카드: 640~720px

텍스트 간 여백은 넉넉하게 설정하여 숨 쉴 공간 확보

7️⃣ 절대 수정 금지 항목

다음 항목은 절대 건드리지 말 것:

가로 → 세로 스크롤 구조

슬로건 이전/이후 섹션

헤더 투명/불투명 로직

이미지 애니메이션

IntersectionObserver 기준점

8️⃣ 최종 목표 요약 (이해 체크)

이 수정의 최종 목표는:

사용자는
→ 처음엔 단 하나의 문장만 인지하고
→ 스크롤을 통해 천천히 다음 메시지를 “발견”하며
→ 마지막에 병원의 원칙을 차분하고 신뢰감 있게 받아들이는 경험을 하게 만드는 것이다.

과한 연출 없이,
정제된 속도와 간격으로 ‘급 차이’를 느끼게 할 것.
