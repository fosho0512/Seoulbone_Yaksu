# Seoul Bone Rehab Clinic Website

## Overview
A static website for Seoul Bone Rehabilitation Medicine clinic. This is a cozy, minimal-themed website built with pure HTML, CSS, and JavaScript.

## Project Structure
- `index.html` - Main HTML page with all the page structure
- `style.css` - Styling with CSS custom properties for theming
- `script.js` - Interactive functionality (menu, modals, animations)
- `data.js` - Content data for dynamic sections
- `images/` - All images used on the site

## Technology Stack
- Static HTML/CSS/JavaScript
- No build system required
- Google Fonts (Manrope, Playfair Display)
- Font Awesome icons

## Running the Project
The project is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment serving from the root directory.
